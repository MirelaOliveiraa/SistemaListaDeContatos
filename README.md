# SistemaListaDeContatos

#### Sobre: 
Challenge desenvolvido usando SqlServer e C#!

## 

#### Tecnologias:
Frontend: Bootstrap <br />
Backend: C#

##

#### Instalação:
> clone o repositório <br />
> abra o projeto  <br />
> exeute Ctrl+F5 <br />
